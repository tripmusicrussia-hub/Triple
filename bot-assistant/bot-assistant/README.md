# Деплой на Railway

## Шаг 1 — GitHub
1. Зайди на github.com → New repository → назови "my-bot"
2. Загрузи все файлы из этой папки (кроме README)

## Шаг 2 — Railway
1. Зайди на railway.app → Login with GitHub
2. New Project → Deploy from GitHub repo → выбери "my-bot"
3. Дождись деплоя

## Шаг 3 — Переменные окружения
В Railway → твой проект → Variables → добавь:

| Переменная        | Значение                  |
|-------------------|---------------------------|
| BOT_TOKEN         | токен от BotFather        |
| ADMIN_ID          | твой Telegram ID          |
| GROQ_API_KEY      | твой Groq ключ            |
| CHANNEL_ID        | ID канала с битами        |
| CHANNEL_LINK      | ссылка на канал           |
| SAMPLE_PACK_FILE_ID | file_id сэмпл пака      |
| WELCOME_TEXT      | приветственный текст      |
| CATALOG_INTRO     | текст перед каталогом     |

## Команды бота
- `/my` — показать сводку (баланс, задачи)
- Любой текст — AI ассистент
- Голосовое сообщение — расшифрует и ответит

## Примеры
- "Потратил 500 на такси"
- "Пришла зарплата 50000"
- "Задача: купить продукты до пятницы"
- "Сколько я потратил за последнее время?"
- "Запомни контакт Иван +79161234567"
