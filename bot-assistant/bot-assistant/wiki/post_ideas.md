# Post Ideas — бэклог тем для текстовых рубрик

Формат: `- [ ] тема` — невостребованная. `- [x] тема` — использована.
Бот берёт случайную невостребованную из нужной рубрики и помечает.

## Вт — Trick of the week (одна фишка в миксе)

> Личный приём: «раньше делал X → теперь Y → слышно здесь». 3-5 строк.
> Одна вещь. Без оценок что «все делают не так».

- [ ] sidechain низа мелодий от баса — теперь они не дерутся с 808
- [ ] Decapitator в режиме E на 808 — 10-15%, больше присутствия
- [ ] дубль хай-хетов через RC-20 — винтажная пыль сверху
- [ ] Sausage Fattener на перкуссию-bus — склеивает в одно тело
- [ ] Little Radiator на вокал-чопе — тембр из 70-х
- [ ] копия 808 октавой выше, -20 дБ — читаемость в маленьких колонках
- [ ] Pro-Q 3 dynamic EQ на 200 Hz у 808 — убирает гудёж только когда бьёт
- [ ] пан хай-хетов L15/R15 — стерео без imager-артефактов
- [ ] reverse reverb перед snare — саспенс на удар
- [ ] Saturn 2 в Tape на басу — ламповая жирнота
- [ ] CLA-76 на snare — быстрая атака, Нью-Йорк-звук
- [ ] сайдчейн от kick на reverb-send — чище низ
- [ ] HPF 30 Hz на всём кроме kick и 808 — место в низу
- [ ] Zenology preset «dark pad» в минус октаву — готовая атмосфера

## Вс — Low & dark (подпись звука)

> DT 880 как setup можно упоминать.

- [ ] 808 с двумя слоями saturation — один в теле, второй в гармониках
- [ ] sub-bass до 30 Hz — почему убираю выше HPF
- [ ] DT 880 помогают услышать где 808 режет с басом — правлю по наушникам
